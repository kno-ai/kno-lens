# Changelog

## 0.1.0

Initial release.

- Live activity tracking for in-progress turns
- Turn summaries with file edits, commands, errors, and searches
- Auto-connect to active Claude Code sessions
- Session picker for manual session selection
- Search across turn prompts and activity labels
- Category filtering (edits, commands, errors, search, agents)
- VS Code theme integration (dark, light, high contrast)
